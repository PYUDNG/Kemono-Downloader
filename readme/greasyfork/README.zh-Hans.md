# Kemono Downloader

一个现代化的Kemono下载器用户脚本，提供美观的UI界面、多种下载器和高度自定义功能。

## ✨ 功能特性

### 🎨 现代化界面
- 基于PrimeVue和Tailwind CSS的现代化UI设计
- 响应式布局，支持桌面和移动端
- 深色/浅色主题支持
- 多语言支持

### 📥 下载功能
- 批量下载支持
- 多种下载器选择
- 智能文件名处理
- 下载进度显示
- 下载任务管理

### ⚙️ 自定义设置
- 不同的下载器自定义
- 自定义文件名（支持创建目录结构）

### 🔧 技术特性
- 模块化架构设计
- 类型安全的TypeScript开发
- 现代化的Vue 3 Composition API
- 自动化的构建流程
- 开发服务器支持HTTPS
